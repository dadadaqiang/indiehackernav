## 🛠 使用

直接打开 `index.html` 即可，所有卡片点击可跳转。

### 本地预览

```bash
cd indiehacker-nav
python3 -m http.server 8765
# 浏览器访问 http://localhost:8765
```

### 在线部署（GitHub Pages）

1. 仓库 Settings → Pages
2. Source 选 **Deploy from branch** → `main` → `/ (root)`
3. 访问 `https://dadadaqiang.github.io/indiehackernav/`
